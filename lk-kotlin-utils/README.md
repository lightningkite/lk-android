# Module lk-kotlin-utils

[Return to Index](../)

## Package description

Various useful extension functions

## Gradle Inclusion

Add the repository:

```
repositories {
    maven {
        url "https://dl.bintray.com/lightningkite/lk-kotlin"
    }
}
```

Include the desired libraries:

```
dependencies {
    implementation "lk-kotlin:lk-kotlin-utils:[version]"
}
```
