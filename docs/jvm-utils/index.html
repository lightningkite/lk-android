Package description

Various useful extensions depending on the JVM. Includes things like stream manipulation, asynchronous execution, date reading, files, and randomness.

Gradle Inclusion

Add the repository: 

repositories {
    maven {
        url "https://dl.bintray.com/lightningkite/com.lightningkite.kotlin"
    }
}
```

Include the desired libraries:

```
dependencies {
    implementation "com.lightningkite.kotlin:utils:[version]"
    implementation "com.lightningkite.kotlin:jvm-utils:[version]"
}
```
